 PERMINTAAN BARU: foto produk di-host BACKEND + bisa diedit dari Admin Site
==============================================================================
 Fitur: Admin upload/ganti foto menu dari Admin Site -> foto tampil di Buyer
 Site DAN otomatis dikirim chatbot saat pelanggan tanya detail kue. (Jalur
 kirim foto di chatbot SUDAH jadi & sudah support skema path relatif ini;
 kolom products.image_url juga sudah ada.)

 Keputusan tim: foto di-serve backend sendiri (bukan Cloudinary/SaaS).

 ATURAN PENTING — SIMPAN PATH RELATIF, BUKAN URL ABSOLUT:
   Kolom image_url diisi HANYA path:  /static/products/12.jpg
   JANGAN http://localhost:8001/... atau http://backend:8001/... — alamat
   host itu beda-beda tergantung pemanggil (browser vs container wwebjs vs
   production). Tiap konsumen mem-prefix base URL-nya sendiri:
     - chatbot  -> BACKEND_BASE_URL + path   (sudah diimplementasi)
     - Buyer/Admin Site -> base URL API + path (kabari Ainaya)

 Yang dikerjakan:
   1) Serving statis (1 baris di app/main.py):
        app.mount("/static", StaticFiles(directory="static"), name="static")
      (folder static/products/ dibuat sekali; di deployment nanti folder ini
       di-mount volume + ikut backup.)
   2) Endpoint upload (aksi Admin/Owner, JWT):
        POST /products/{product_id}/image     (multipart/form-data, field "file")
          - Validasi: content-type image/jpeg|png|webp, ukuran <= 5 MB.
          - Simpan ke static/products/{product_id}.{ext}
            (nama file = id produk -> ganti foto otomatis menimpa; tambahkan
             query versi kalau mau cache-bust, mis. ?v=timestamp).
          - Set products.image_url = "/static/products/{product_id}.{ext}"
          - Response 200: ProductOut (image_url terisi path relatif).
   3) (Opsional, 1 baris) image_url: Optional[str] di ProductUpdate — berguna
      buat menghapus foto (set null) atau koreksi manual.

 SELESAI JIKA: upload foto via POST /products/12/image ->
   GET  {base}/static/products/12.jpg  tampil di browser, dan
   GET  /products/12 -> image_url == "/static/products/12.jpg", lalu
   tanya "cake 10cm kayak gimana?" ke chatbot -> foto masuk di WhatsApp.
